## What this module does

Every heading is a link to itself — click one and copy the address to share
a deep link. Cross-reference any section from anywhere, like this link to
[combining items](#combining-items) in the next chapter.

## Anchors

Headings get invisible anchor aliases in legacy shapes, so links like
`#what_this_module_does` keep working alongside Hugo's own
`#what-this-module-does`.
